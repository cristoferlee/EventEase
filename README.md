# EventEase Blazor App

EventEase is a Blazor WebAssembly project created for the Coursera guided lab. The app demonstrates foundational Blazor components, routing, validation, state management, performance optimization, and event attendance tracking.

## Activity 1: Foundational Components

Microsoft Copilot helped generate the foundational Event Card component with event fields such as name, date, and location. Copilot also assisted with data binding and the initial structure of the app.

## Activity 2: Debugging and Optimization

Copilot helped identify and fix issues related to input validation, routing errors, and performance. The app was improved with validation patterns, better route handling, `NotFound` routing, `@key`, and optimized event rendering.

## Activity 3: Advanced Features

Copilot helped implement a Registration Form with validation, user session state management using `UserSessionService`, and attendance tracking logic to monitor which events the user is attending.

## Features

- Event Card component
- Event list and event details pages
- Registration form with validation
- User session tracking
- Attendance tracking
- Routing and NotFound handling
- Performance optimization using Blazor best practices
